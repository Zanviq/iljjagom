## 저장 방법
1. 각 화면을 캡처합니다(가능하면 mock 데이터 말고 실제 흐름 화면).
2. 아래 표의 **파일명**으로 저장해 이 폴더(`documents/screenshots/`)에 넣습니다.
3. 형식은 `.png` 를 권장합니다(투명 불필요). 용량이 크면 가로 1600px 이하로 줄여 저장합니다.
4. 커밋하면 GitHub README 에 바로 표시됩니다.
