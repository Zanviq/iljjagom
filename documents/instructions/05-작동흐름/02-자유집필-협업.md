# 자유집필 협업 (기·승)

> 기·승(free) 챕터는 AI가 통째로 써 주는 것이 아니라 **학생과 AI가 한 문단씩 함께 짓는다.** 화면은 좌(본문 누적) + 우(AI 대화) 2단.

## 상호작용 모델 — 한 문단 = 한 협업 턴
```
[우: AI 대화]                          [좌: 본문]
AI: 이 이야기 어떻게 시작할까?
학생: 숲에서 길 잃은 토끼로 시작해   ──생성──▶  ¶1 "깊은 숲, 작은 토끼가…"
AI: 좋아! 이제 토끼는 어떻게 돼?
학생: 용을 만나서 친구가 돼
   ↑ AI가 흐름·주제 점검(assess_flow)
AI: 물론 그것도 좋지! 근데 이건 어때?
    토끼가 먼저 무서워하다 친해지면, 앞 문단(길 잃음)과
    더 잘 이어지고 '용기' 수업 주제도 잘 드러나거든.   ← 지도(coaching)
학생: 오 그게 낫겠다  (accept) ──생성──▶  ¶2 "토끼는 처음엔 떨었지만…"
... (문단이 충분히 쌓이면) chapterComplete → 중간활동
```

## 규칙
1. **한 학생 응답 = 한 문단 생성.** 통짜 생성 금지.
2. AI는 매 턴 끝에 **진행 질문 1개**를 던진다.
3. 학생 의도가 (a) 직전 문단과 어색하거나 (b) 학급 학습목표에서 벗어나면 → **지도**한다: 긍정("물론 그것도 좋지!") + 대안 + **근거**. 학생이 받아들이면(accept) 대안으로, 고수하면 학생 의도대로 생성(아동 주도성 존중).
4. **결말/secretArc는 기·승 협업에서 절대 노출하지 않는다.**

## AI 엔진
- `chat.assess_flow` — 흐름·주제 점검 → `generate` 또는 `coach` 결정 + 근거.
- `writer.write_paragraph` — 학생 의도 + 직전 문단 + RAG로 한 문단 생성(결말 비노출).
- `chat.next_paragraph_question` — 다음 진행 질문.
- 출력은 본문 정제(마크다운·머리말 제거)와 출력 안전을 통과한다.

## AI 지도 강도 (coachingLevel)
협업 턴(`collab_turn`)은 학급 설정의 **AI 지도 강도**(`policy.resolve_coaching_level` = 학급 설정 > 기본 `light`)로 `assess_flow` 를 호출해 간섭 정도를 조절한다.
- `off` — 점검을 생략하고 항상 생성(지도 없음).
- `light`(기본) — 흐름만 점검. 주제 일탈은 아이의 창의성으로 허용.
- `standard` — 흐름 + 학습 주제 점검.

교사는 학급 설정 화면의 "AI 지도 강도" 셀렉터로 이 값을 바꾼다(→ `03-기능/02-교사.md`).

## 여는 말 (동적)
자유집필 진입 시 곰 작가의 여는 말은 고정 문구가 아니라 Bible 인물명·장 번호로 동적 생성된다(`CollabState.openingQuestion`). 2장 이상이면 "지금까지 이어온 …의 이야기, N장에서는?"처럼 장 연속성을 표현한다. 화면 상단에는 "현재 장 / 전체 장"이 표시된다.

## 데이터 흐름
- 문단 → `chapter_paragraphs(chapter_id, seq, body, source)`. 대화 턴 → `writing_turns(role, kind, content, paragraph_id)`.
- 문단을 `seq` 순으로 이어 `chapters.body`로 재조립 → 독서·RAG·학습활동은 기존대로 `body`를 읽는다(하위호환).
- 새로고침 복원: `GET /books/{id}/chapters/{n}/collab` → 좌(문단)·우(대화) 동시 복원.

## API
- `POST /books/{id}/chapters/{n}/collab` `{ message, accept? }` → `CollabReply`(paragraph 또는 coaching + question + chapterComplete). 1차는 단일 응답(POST), 추후 문단 SSE 격상 가능.
- free 챕터에서만 유효(guided는 409).

## 전·결과의 경계
전·결(guided)은 AI가 결말을 비밀로 펼치는 단계라 협업 대상이 아니다. 기·승 협업이 끝나면 중간활동을 거쳐 전·결이 시작된다.
