# 화면 가이드

> 역할별 셸과 화면 구성. 기능적 구성요소 중심(상세 기능은 `03-기능` 참고).

## 공통 셸
| 환경 | 셸 |
| --- | --- |
| 학생 | 상단 바(워드마크·학급 배지·로그아웃) + 우하단 **곰 작가 FAB** + 우측 **AI 드로어**(메인 화면에만) |
| 교사 | 좌측 **고정 사이드바**(학급 셀렉터·발제·대시보드·학생·안전·게시판; 미처리 안전엔 danger 배지) + 콘텐츠 헤더 |
| 관리자 | **아이콘 레일** + 라이브 상단 상태바(시계·"SYSTEMS NOMINAL") + 다중 컬럼 지표 그리드 |

## 학생 화면
| 화면 | 구성 |
| --- | --- |
| 홈 `/home` | 인사 헤더 · "이어 읽던 이야기"(진행률 카드) · "새 이야기 시작"(발제 카드) |
| 기획 `/plan` | AI 대화 채팅 + 인물 카드 미리보기 + "이야기 시작하기" |
| 기·승 협업 `/write` | **좌: 본문 문단 누적 / 우: AI 대화**(지도 시 "이대로/제안대로" 버튼) |
| 중간활동 | 중간 퀴즈/독후감(필수) |
| 독서 `/read` | 장 표시 · (유도)삽화 선노출+능동질문 → 탭하면 본문 스트리밍 · 단어 도움 팝오버 · "이 장의 낱말" · 다음 장 · (자유)이야기 고치기 |
| 학습 `/learn` | 낱말 카드 · 퀴즈(정오답 피드백) · 감정 곡선(학생 입력) · 인물 편지(인물 선택) · 독후감 · "학급에 발표하기" |
| 게시판 | 학급 발표 목록·상세(표지·요약·감정곡선·편지) |

## 교사 화면
| 화면 | 구성 |
| --- | --- |
| 학급 `/classes` | 학급 카드(이름·학생수·가입코드·링크), 학급 만들기 |
| 발제 | 주제·학습목표(추가/삭제)·평가·언어·상태 폼 + 낸 발제 목록(클릭→학생 제출) |
| 대시보드 | 학급 선택 · 요약 카드 · 학생별 진척 표 · 활동 추세 |
| 학생 열람 | 본문·기획·협업·Bible 읽기 전용 |
| 안전 검토 | safety_flags·held 편지 목록/상세/승인·반려 |
| 게시판 승인 | 발표 승인 큐(승인/반려) |

## 관리자 화면
| 화면 | 구성 |
| --- | --- |
| 콘솔 9탭 | 개요·세션·사용자·대화·토큰·안전·설정·알림·백업 |
| 드릴다운 | 지표/목록 클릭 → 사용자 → 책 → 단계 → 세션 → 스텝 |
| SessionInspector | 우측 패널: 프롬프트 스냅샷·스킬·대화 전문·토큰·이미지 생성 과정 |
| 설정 | 선택형(드롭다운/토글/스텝퍼) — 모델·주기·안전강도 enum |

## 원칙
- 학생 화면은 한 화면 한 액션·큰 글자·읽기 몰입. 교사는 정보 밀도·표. 관리자는 계기판·실시간.
- 모든 화면은 새 디자인 토큰을 쓰며 색/크기 하드코딩·이모지를 쓰지 않는다.
- 디자인-중립 콘텐츠 명세(무엇이 담겨야 하는가)는 `documents/for-claude/planning-designing/서비스-및-페이지-구성.md` 참고.
