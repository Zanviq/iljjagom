# 디자인 시스템

> 세 역할 환경이 **하나의 토큰 백본**을 공유하고 스코프 클래스로 리-테마한다. 학생=base(`:root`), 교사=`.theme-teacher`, 관리자=`.theme-admin`. 이모지는 전면 금지(Lucide 라인 아이콘).

## 1. 세 환경
| 환경 | 이름 | 무드 | 색 | 형태/모션 |
| --- | --- | --- | --- | --- |
| 학생 | Reading Room | 따뜻·놀이 | 크림 Paper · 허니 Amber · 세이지 | 둥근 카드(28px) · 부드러운 그림자 · 바운스 |
| 교사 | Workbench | 명료·실무 | 슬레이트 · 인디고 · 틸 | tight 반경(10px) · 플랫 보더 · 빠른 모션 · 표 |
| 관리자 | Control Room | terse·계기판 | 딥네이비 Abyss · 시안/라임 · mono | glow 경보 · 밀집 그리드 |

서브트리를 `.theme-teacher`/`.theme-admin`으로 감싸면 surface·text·primary·radius·shadow·motion·폰트가 통째로 리-맵된다. 학생은 래퍼가 필요 없다.

## 2. 타입
| 폰트 | 용도 |
| --- | --- |
| Newsreader (serif) | 워드마크 · 디스플레이 · **학생 본문 21px / 1.85** |
| Pretendard (sans) | 모든 UI · 한글 본문 · 라벨 |
| IBM Plex Mono | 수치 · ID · 관리자 콘솔 |

## 3. 토큰 (요지)
- **색**: 3중립 패밀리(Paper/Slate/Abyss) + 공유 액센트(Amber=브랜드·학생 primary, Sage, Indigo=교사, Cyan/Lime=관리자 신호) + status(green/gold/red). 의미 토큰(`--bg`·`--surface`·`--text-1`·`--primary`·`--line` 등)을 컴포넌트가 참조하고 테마가 리-맵.
- **간격**: 4px 그리드. 터치 최소 44px, **학생 56px**. 폭: 읽기 38rem · 셸 64rem · 관리자 90rem.
- **반경**: xs6/sm10/md16/lg22/xl28/pill. 학생 round, 교사·관리자 tight. 보더 1.5px.
- **그림자**: 학생 soft/warm · 교사 crisp · 관리자 depth + glow(cyan/lime/red).
- **모션**: 학생 `ease-bounce`·느린 듀레이션·`ijg-rise/pop`·스트리밍 캐럿, 교사/관리자 `ease-snap`·빠름·`ijg-pulse`. `prefers-reduced-motion` 준수.

## 4. 아이콘 / 카피
- **Lucide** 라인 아이콘(~2px). **이모지 일절 사용 안 함.** 브랜드 마크는 `book-heart` 로고 타일 + Newsreader 워드마크.
- **역할별 보이스**: 학생=따뜻·놀이·쉬운 말("이 주제로 새 책 만들기"), 교사=명료·실무 명사("완독률"·"학생별 진척"), 관리자=terse·계기판·mono("SYSTEMS NOMINAL").

## 5. 컴포넌트 (14)
- core: Button · Badge · Chip · Avatar · Icon
- surface: Card · StatCard · ProgressBar
- forms: Field · Input · Textarea · Select · Checkbox · Switch
- feedback: ChatBubble · EmptyState

## 6. 구현 (프론트)
디자인 토큰 CSS는 `frontend/app/design/tokens/*`로 도입되고 `globals.css`에서 import + Tailwind v4 `@theme` 브리지. 폰트는 next/font(Newsreader·IBM Plex Mono) + Pretendard 패키지. 교사/관리자 라우트 그룹 layout이 테마 클래스를 씌운다. 원본 디자인 시스템 정의는 `documents/for-claude/attached/new-design_version2/`.
