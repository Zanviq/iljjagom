# 일짜곰 Frontend

**Next.js 15 (App Router) + TypeScript** 기반 웹 프론트엔드.
웹에 최적화된 형태로 제작하며, 학교(브라우저 접근)·가정 모두에서 사용 가능하도록 한다.

## 기술 스택
- Next.js 15 (App Router), React, TypeScript
- Tailwind CSS
- Supabase 클라이언트(@supabase/ssr) — 인증·실시간
- SSE로 AI 본문 글자 단위 스트리밍 수신

## 화면 골격 (계획)
- 학생: 기획(자유 집필 대화) → 독서(유도 집필, 삽화 선노출) → 학습 활동(어휘 카드·퀴즈·인물 편지·감정 곡선)
- 교사: 발제(주제·학습 목표·평가) 설정, 학급 대시보드·리포트
- 관리자: 운영 모니터링
